    Simulation

    Um die Simulation zu starten, wird das Hauptskript "main.py" ausgeführt, welches alle Informationen und Funktionen aus den anderen Skripten zusammenführt.

    Hier werden die notwendigen Parameter definiert, die das PV-Modell charakterisieren. Dafür sind zwei elementare Funktionen der pvlib-python wichtig.

    PVSystem

    Mit dem PVSystem-object werden die Parameter der PV-Anlage festgelegt. Dazu zählen u.A. die Modul- und Wechselrichtereigenschaften, die Anzahl der Module, die Ausrichtung, das Temperaturmodell usw. Es können hierbei auch mehrere Strings als arrays definiert werden, die an einem Wechselrichter angeschlossen werden.

    ModelChain

    Das ModelChain-object enthält das PVsystem. Hierbei wird die Location bestimmt, welche mithilfe des bereitgestellten Breiten- und Längengrads aus der "config.py" Datei definiert wird. Außerdem können hier noch Verlustmodelle des PVsystems definiert werden.

    Um die Simulation durchzuführen, werden die Wetterdaten benötigt und als Parameter in der "run_model" Funktion übergeben.

    Ausgabe der Simulation

    Eine Grafik und eine Tabelle der monatlichen AC-Erträge (in kWh) der einzelnen Wechselrichter werden als ".png" und ".csv" Datei im angegebenen Ordnerpfad abgelegt. Wenn in der "config.py" kein Ordnerpfad angegeben ist, werden die Dateien dort gespeichert, von wo aus das "main.py" Skript ausgeführt wird (siehe Konfiguration).
